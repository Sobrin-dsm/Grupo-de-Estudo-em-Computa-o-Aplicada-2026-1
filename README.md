# Grupo-de-Estudo-em-Computa-o-Aplicada-2026-1
O Grupo de Estudo em Computação Aplicada é uma atividade formativa voltada a alunos do
curso de Sistemas de Informação que desejam aprofundar sua base técnica e científica em
programação, análise de dados e inteligência artificial.

Em 2025, o grupo trabalhou os fundamentos de Algoritmos e Lógica de Programação (2025/1) e
Estrutura de Dados (2025/2). Em 2026/1, o grupo avança para um novo eixo:

# Fundamentos de Ciência de Dados e Análise Multivariada Aplicada com Python.

Este eixo é estratégico para:
• desenvolver a capacidade de trabalhar com dados reais e multivariados;
• construir a base conceitual para IA, BI e Sistemas de Apoio à Decisão;
• e preparar alunos para projetos que envolvam análise de dados complexos.

Do ponto de vista computacional, muitos problemas modernos de pesquisa (inclusive imagens)
podem ser vistos como grandes matrizes de dados multivariados. O grupo trabalhará
exatamente essa forma de pensar.
